# Welcome to my land

A simple 3D game built with Three.js.

## Live Demo

Play the game: [https://rivedulac.github.io/3d-world/](https://rivedulac.github.io/3d-world/)

## Game Controls

- W - Move Forward
- S - Move Backward
- ← - Rotate Left
- → - Rotate Right
- ↑ - Look Up (Max 60°)
- ↓ - Look Down (Max 30°)
